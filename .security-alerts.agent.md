# Security Alerts Agent

This agent consolidates all security scanning alerts from CodeQL, package vulnerabilities, and Trivy container scanning into a unified alert system with comprehensive reporting.

## Features

- Aggregates security findings from multiple sources
- Creates GitHub issues for security vulnerabilities
- Generates comprehensive security reports
- Tags and labels all findings for easy tracking
- Provides remediation guidance

## Execution

The agent runs as part of the CI/CD pipeline after:
- CodeQL security analysis
- Package vulnerability scanning
- Trivy container image scanning

## Report Generation

Generates the following artifacts:
- `security-alerts.detailed.md` - Comprehensive security alert report
- GitHub Issues - Individual issues for each vulnerability type
